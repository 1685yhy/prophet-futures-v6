# Prophet Futures v10 — 使用手册

## 环境准备（一次性）

```bash
cd /home/a/prophet_futures/prophet_futures
source .venv/bin/activate
```

## 每日操作（10秒）

```bash
python run.py
```

输出三种情况：

### 情况1：无信号
```
📊 LH: 今日无信号 → 不操作
📊 JM: 今日无信号 → 不操作
```

### 情况2：有信号
```
🟢 JM 做多信号！
  入场: 1,350   止损: 1,320   止盈: 1,410
  手数: 2手     盈亏比: 1:2.0
  ⚡ 次日开盘做多，设好止损止盈
```

### 情况3：ML不确认
```
⚠️ 规则信号=LONG | ML置信度=45% < 50%
决策: 放弃 — ML不确认此信号
```

---

## 执行规则

| 规则 | 说明 |
|------|------|
| 信号出现 | 次日开盘执行 |
| 止损 | 立即设好，不修改 |
| 止盈 | 到了就平，不贪 |
| 连亏3笔 | 当月停手 |
| 月亏5% | 立刻停手 |
